# Budget Tracker

A personal finance web app designed for college students to track income and expenses, set savings goals, and visualize monthly spending.

Built with React, Node.js, Express, PostgreSQL

---

## Features

- Add and delete income and expense transactions
- Category labels: Food, Transport, Housing, Entertainment, Health, Education, Shopping
- Spending breakdown by category with visual progress bars
- Monthly budget tracker with remaining balance
- Savings goals with progress tracking
- Monthly spending chart across 6 months
- Filter transactions by type or category

---

## How to Use

1. Open index.html in your browser
2. Go to Transactions and add your income and expenses
3. View your Dashboard for a full summary
4. Check Goals to track your savings progress

---

## Tech Stack

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js, Express
- Database: PostgreSQL
- Deployment: Render
